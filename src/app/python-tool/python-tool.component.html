Tracker tool for warframe items

ATTENTION!

This tool is not directly endorsed by Digital Extremes or its team. It will not change anything in your game, however, we cannot guarantee incompatibility with software on your machine (such as anti-virus) or with game anti-cheats, we recommend that after use, you close the tool to continue playing. USE AT OWN RISK

This tool uses Python as programming language, you can get more info about it here: Python page

To scan images, this tool uses tesseract-ocr, Tesseract OCR is an open source project focused on image scanning. Check more info about Tesseract here: Tesseract OCR

This tool code can be checked here: Item tracker python code

~100MB

IMPORTANT

Dark colors, such as black, navy blue, brown, gray, among other dark colors, can impact character identification. We suggest that during prints, for greater precision choose to paint your warframe/weapons completely white.

HOW TO USE IT

1 - DOWNLOAD RAR, and extract IT on some folder

2 - Execute api.exe, and check CMD/Console for some info

You need to use CTRL+PRINTSCREEN to print images

You need to use CTRL+ESC to close api running

Alternatively, you can manually take screenshots as you wish, and manually place the images in the API folder.

Alternatively, you can terminate the api by closing the CMD/Console

3 - IN FIRST PRINT, it may not work automatically, opening the TK framework, just click on the icon and the following times, it will work automatically.

4 - ALL IMAGES are saved next to the API executable, if you wish, you can manually paste your prints there (important if you want to re-execute a routine in the future)

5 - VISUAL PREVIEW of the print is informed in the CMD/Console

6 - WE RECOMMEND open your warframe inventory, through "SHOW PROFILE" in your player icon, then Inventory -> Sort by Mastery, to show all mastered items first

7 - For more text accuracy, you can capture just the item name.

8 - Click in Start Scan

9 - If everything goes as planned, all items will be compared with the images provided.
