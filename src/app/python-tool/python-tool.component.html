This tool is not directly endorsed by Digital Extremes or its team. USE AT OWN RISK

- This tool uses Python as programming language, you can get more info about python here: Python page

- To scan images, this tool uses tesseract-ocr, Tesseract OCR is an open source project focused on image scanning. Check more info about Tesseract here: Tesseract OCR

- This tool, referred to from now on as API, can have the code checked here: API for item tracker python code

- So far, it will only work in Windows.
STEP {{current}} of 7

Download de API .rar, it will contain the python application (api.exe) + TesseractOCR (folder)

~100MB

Choose a location and extract the contents of the rar, the TESSERACT folder must be in the same location as api.exe:

Run api.exe, and check CMD/Console for some info if you want.

You need to use CTRL+PRINTSCREEN to print images

You need to use CTRL+ESC to stop and close api.

Alternatively, you can manually take screenshots as you wish, and manually place the images in the API folder.

Alternatively, you can terminate the api by closing the CMD/Console

IN FIRST PRINT USE, it may not work automatically and will open the TK framework on taskbar instead, just click on the icon and next times, it will work automatically.

ALL IMAGES are saved in the same location as the API.exe, if you wish, you can manually paste your prints there (important if you want to re-execute a routine in the future without capturing all prints again)

A VISUAL PREVIEW of the print is informed in the CMD/Console.

WE RECOMMEND open your warframe inventory, through "SHOW PROFILE" in your player icon, then Inventory -> Sort by Mastery, to show all mastered items first

Some tips for better recognition:

- Avoid dark colors, such as black, brown, purple, Tesseract may confuse dark backgrounds as "characters", prioritize painting your warframe/operator completely clear for captures

- The game in English will be better tracked, since the imported data is in English

- For more text accuracy, or to avoid painting your warframes, you can capture just the item name. It's more work, but more accurate.

Now, click in "Start Scan" on website

If everything goes as planned, all items will be compared with the images provided.
